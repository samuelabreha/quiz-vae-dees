# Règle d’entraînement – Méthode universelle de réponse orale

## Principe

Pour toute question (quiz ou micro-quiz), la réponse doit être formulée selon la structure suivante :

Contexte  
Problématique  
Objectifs  
Actions  
Posture  
Réseau  
Évaluation  
Référentiel

## Règle

Même si certaines étapes sont très courtes, les 8 éléments doivent apparaître dans la réponse.

### Réseau (minimal acceptable)

Le réseau peut être minimal, par exemple :

- transmission à l’enseignant
- coordination avec l’équipe
- échange avec la famille

### Évaluation (format minimal)

L’évaluation peut être formulée comme :

- effet observé + ajustement professionnel

### Référentiel (contenu attendu)

Le référentiel doit contenir :

- DC principal + loi ou repère théorique si pertinent

## Objectif pédagogique

Tous les quiz servent désormais à entraîner cette structure narrative professionnelle pour l’oral VAE DEES.

Ce fichier sert uniquement de règle d’utilisation du système de révision.
