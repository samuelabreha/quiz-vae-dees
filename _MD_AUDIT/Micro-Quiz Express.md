## Conseils d'usage (agent préparateur DEES)

- Commencez par **1 micro-quiz** pour activer la mémoire.  
- Enchaînez avec **1 quiz complet** pour simuler l'oral.  
- Répondez toujours avec le fil : **DC -> cadre -> situation -> actions -> analyse -> généralisation**.  
- Gardez un ton clair, posé, professionnel : vous démontrez une compétence, vous ne racontez pas seulement un souvenir.
